### 자바에 대한 공부 기록(git-flow연습 적용)

### git-flow 적용 방법

 + **feature: 코드연습이 끝난 후(기능추가)**
 + **develop: 코드연습과 블로그 업로드가 끝난 직후**
 + **release: 코드연습과 블로그업로드가 끝나고 하루 뒤**
 + **master: 코드연습과 블로그 업로드가 끝나고 확인까지 마쳤을 때-최종**
 + **hotfix: 혹시라도 에러가 발생할 경우**
